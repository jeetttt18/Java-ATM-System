# Java ATM Management System

A desktop-based ATM Management System developed using **Java Swing and MySQL**.

## Features
- User Signup
- User Login
- Deposit Money
- Withdraw Money
- Check Balance
- Account Statement
- Transaction History

## Technologies Used
- Java
- Swing (GUI)
- JDBC
- MySQL
- Git

## Database Tables
users  
transactions
